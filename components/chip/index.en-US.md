## API

### Chip

| Property | Description                                      | Type              | Default |
| -------- | ------------------------------------------------ | ----------------- | ------- |
| size     | Chip size, default size is regular               | `small`           | -       |
| checked  | Chip is checked/unchecked                        | boolean           | false   |
| onChange | Callback executed when Chip is checked/unchecked | (checked) => void | -       |
